# Paylocity Bug List Challenge 🐛

Welcome to the **Paylocity Bug List Challenge** repository! This project contains comprehensive documentation of issues, inconsistencies, and defects discovered during the testing of the Paylocity Benefits Dashboard.

## 📂 Repository Contents

The findings have been categorized into two main areas, which are documented in the detailed PDF reports included in this repository:

* **`API Bugs on Paylocity Benefits Dashboard.pdf`**
    * Contains backend, data-handling, and API-related issues.
    * Focuses on data validation, request/response payloads, and server-side logic errors.

* **`UI Bugs on Paylocity Benefits Dashboard.pdf`**
    * Highlights front-end, visual, and User Experience (UX) issues.
    * Focuses on layout inconsistencies, responsive design flaws, typos, and visual component states.

## 🎯 Objective

The purpose of this repository is to demonstrate thorough Quality Assurance (QA) practices. The reports aim to provide clear, actionable feedback for developers by detailing:
* Clear descriptions of the bugs.
* Steps to reproduce each issue.
* Expected behavior vs. Actual behavior.
* Severity/Priority levels for each finding.
